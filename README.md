# Flutter To-Do App

A simple To-Do List mobile application built using Flutter and Dart.

## Features
- Add tasks
- Mark tasks as complete
- Delete tasks
- Simple UI

## Tech Stack
- Flutter
- Dart
- Material Design

## Getting Started

Clone the repository:

git clone https://github.com/komalsao/flutter-todo-app.git

Run the project:

flutter pub get
flutter run
